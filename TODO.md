# validate: (ok)

1. 开始节点至少一条 next 边
2. 结束节点至少一条 prev 边
3. 其余节点至少一条 next 边 + 一条 prev 边
4. 每个节点自己做校验 , 并根据结果显示 感叹号 在后面
5. 添加 onAddNode / onDeleleteNode 两个事件(调用者收到事件后, 可立即创建空外部节点) ok

# 自动对齐

当两个节点存在 相互依赖 (体现在图上, 就是 A B 节点 互相连接),
这时有两种方案:
方案 1: 节点类型中, 增加 附属节点 属性, 它将与 主节点出现在同一排
